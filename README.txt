
-------------- Batch Render Program -----------------
----------- Moondog Animation Studio ----------------
------------ Created by: Erin Terre -----------------
--------- Last Update: 02 - 24 - 2016 ---------------

This python script creates a simple UI that allows a user to render multiple cameras at once in a UE4 packaged game. 

FEATURES:
	> renders all 6 cameras of a 6 camera rig at once in a UE4 packaged game through the command line
	> organizes renders into folders by camera name
	> renames rendered images by camera name and frame number
	> allows user to select cameras to render in the event of needing to re-render
